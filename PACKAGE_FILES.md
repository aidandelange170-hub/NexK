# DeepSeek AI Application Package Files

This document lists all the files included in the DeepSeek AI application package:

## Core Application Files
- `main.py` - The main AI application with configuration support
- `setup_gui.py` - GUI settings application for API key configuration
- `run_ai.py` - Simple launcher for the main application

## Configuration and Dependencies
- `requirements.txt` - Python dependencies
- `.env.example` - Example environment file

## Installation Scripts
- `install.bat` - Windows installation batch script
- `setup_installer.py` - Python setup script
- `installer.nsi` - NSIS installer script

## Documentation
- `README.md` - Main application documentation
- `INSTALLATION.md` - Installation guide
- `PACKAGE_FILES.md` - This file

## Usage Examples
- `example_usage.py` - Example usage of the AI application

## Build Scripts
- `create_executables.py` - Script to create executables using PyInstaller

## Setup Files
- `setup.py` - Package setup file

All these files are needed to create a complete, distributable package of the DeepSeek AI application with settings functionality.
